## 1. Configure sharding rule

Edit `%SHARDING_PROXY_HOME%\conf\sharding-config.yaml`. More details please reference [Configuration Manual](/07-sharding-proxy/configuration/). 

## 2. Start server

```sh
${sharding-proxy}\bin\start.sh ${port}
```
