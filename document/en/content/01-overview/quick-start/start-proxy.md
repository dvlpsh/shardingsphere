## Quick Start


### Sharding-Proxy

#### 1. Configure sharding rule

Edit `${sharding-proxy}\conf\sharding-config.yaml`. Same format with Sharding-JDBC-Driver's `YAML` configuration. 

#### 2. Start server

```sh
${sharding-proxy}\bin\start.sh ${port}
```
