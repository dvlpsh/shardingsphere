## Encryptor

| *SPI 名称*    | *详细说明*            |
| ------------ | --------------------- |
| Encryptor    | 数据加密算法           |

| *已知实现类*  | *详细说明*             |
| ------------ | --------------------- |
| MD5Encryptor | 基于 MD5 的数据加密算法 |
| AESEncryptor | 基于 AES 的数据加密算法 |
| RC4Encryptor | 基于 RC4 的数据加密算法 |
